# knol docs

Welcome. This folder exists so you can practice editing content while
developing `knol` itself. When you run `bun run dev:server` from the
project root, the sidebar picks these files up automatically.

## Layout

- `README.md` — this file
- `guide/` — how-to content
- `concepts/` — design notes

Try editing any of these and refreshing the browser — you should see
the changes immediately (the server reads from disk on every request).
